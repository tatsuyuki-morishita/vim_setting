# vim_setting

## 使用前提
使用シェルがbashであること。

## Quick Start

   ```
   #ディレクトリ作成 && git clone
   git clone https://github.com/tatsuyuki-morishita/vim_setting.git ~/.vim_setting/
   #シンボリックリンクを貼る等の初期設定ファイル実行
   cd ~/.vim_setting
   bash setup.sh
   #実行したそのセッションのみで実行する必要あり
   source ~/.bash_profile
   ```
